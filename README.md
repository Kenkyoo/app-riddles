# App Riddles

App Riddles es una aplicación web sencilla de acertijos que permite a los usuarios obtener preguntas interesantes y revelar sus respuestas con un clic. Fue creada utilizando tecnologías modernas como **Parcel** para la construcción del proyecto y desplegada en **Vercel**.

## Características

- Obtén acertijos dinámicos desde una API externa.
- Interfaz moderna y amigable con componentes estilizados.
- Respuesta oculta hasta que el usuario decida revelarla.
- Animaciones simples para una experiencia interactiva.

## Tecnologías Utilizadas

- **HTML5**, **CSS3** y **JavaScript** para la estructura, estilos y funcionalidad.
- **Parcel** como empaquetador del proyecto.
- **JQuery** y **JQuery UI** para manejar eventos y animaciones.
- **MDB UI Kit** para estilos y componentes visuales.
- **Vercel** para el despliegue continuo y hosting.
- **API Ninjas** para obtener acertijos dinámicamente.

Uso

    Haz clic en Start para cargar un nuevo acertijo.
    Visualiza la pregunta del acertijo en la tarjeta que aparece.
    Haz clic en Show Answer para revelar la respuesta.

API Utilizada

Se utiliza la API de API Ninjas para obtener los acertijos. Asegúrate de configurar tu clave de API en el código (app.js).
